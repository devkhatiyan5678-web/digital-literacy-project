# **Platforms**  
This task builds familiarity with two categories: coding practice platforms and cloud collaboration tools  

### Part A: Coding Practice  
**HackerRank, CodeChef, or HackerEarth** provides a structured environment to _sharpen coding skills_ through challenges across various domains like **Algorithms, Data Structures, and AI.** Its Integrated Development Environment **(IDE)** allows real-time execution, while automated test cases _ensure code accuracy_. Rank-based leaderboards and skill certifications further help developers **track** progress and prepare for **technical interviews**.

---  

### Part B: Google Workspace Collaboration  
**Google Workspace enhances** coding practice by providing a seamless **real-time** collaborative environment. Teams can use **Google Docs** for collective brainstorming and architectural planning, while **Google Meet** enables synchronous pair programming and code reviews via screen sharing. _Integrated commenting and version history_ ensure clear communication and easy tracking of logic changes.
